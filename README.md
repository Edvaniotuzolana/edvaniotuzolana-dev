# Como utilizar os projetos

Este repositório contém projetos de estudo e prática em programação.

Para utilizar um projeto:

1. Escolha a pasta do projeto desejado.
2. Leia o arquivo README.md dentro da pasta (quando disponível).
3. Siga as instruções de compilação e execução.

- Compilador C (GCC ou Clang)
